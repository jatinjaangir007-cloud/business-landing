## 🎨 Design System

```css

--gold-light: #e8d5b0   /* Hover states */
--dark:       #0a0907   /* Page background */
--dark2:      #13110e   /* Card backgrounds */
--dark3:      #1e1a15   /* Nested surfaces */
--cream:      #f5f0e8   /* Headings */
--warm-gray:  #8a8070   /* Body text */
```

**Fonts:**
- Display / Headings → `Cormorant Garamond` (serif, italic)
- Body / UI → `Jost` (sans-serif, 300–600)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/lumiere-hotel-template.git
cd lumiere-hotel-template
```

### 2. Open in browser

```bash
# No build step needed — just open the file
open demo-website.html
# or double-click the file in your file explorer
```

### 3. Customise for your client

| What to change | Where to find it |
|----------------|-----------------|
| Hotel name & logo text | Search `Lumière` in the HTML |
| Colour scheme | `:root` CSS variables at the top |
| Room names & prices | `#page-rooms` section |
| Contact details | `#page-contact` section |
| Google Maps location | Replace the `iframe src` URL |
| Social media links | `href="#"` on each `.social-link` |
| Fonts | `<link>` tag in `<head>` → swap Google Fonts URL |
